Description

This function copies all of the vertices and edges to which the specified root vertex is connected from graph into result. Also, it copies the vertex and edge properties, either by copying the appropriate contents of get(vertex_all, graph) to get(vertex_all, result) and by copying the appropriate contents of get(edge_all, graph) to get(edge_all, result), or by user-supplied copy functions.

Where Defined

boost/graph/copy.hpp

Then Boost.Parameter's type deduction facilities can bind an unnamed argument to this parameter as long as all of the following are true:

  • The argument's position is the same as that of a parameter whose category is rendered by this document such that the text is of the same color.

Boost.Parameter keyword: boost::graph::keywords::_vertex_copy

Description: This is a Binary Function that copies the properties of a vertex in the original graph into the corresponding vertex in the copy.
Default: A function object that copies the appropriate contents of get(vertex_all, graph) to get(vertex_all, result).
Category: IN
Boost.Parameter keyword: boost::graph::keywords::_edge_copy
Old-style signature: edge_copy(EdgeCopier ec)
Description: This is a Binary Function that copies the properties of an edge in the original graph into the corresponding edge in the copy.
Default: A function object that copies the appropriate contents of get(edge_all, graph) to get(edge_all, result).
Category: IN
Boost.Parameter keyword: boost::graph::keywords::_vertex_index_map
Old-style signature: vertex_index_map(VertexIndexMap i_map)
Description: This maps each vertex to an integer in the range [0, num_vertices(graph)). The argument type must be a model of Readable Property Map. The value type of the map must be an integer type. The vertex descriptor type of the graph needs to be usable as the key type of the map.
Default: get(vertex_index, graph) if the graph has an internal vertex_index property, otherwise dummy_property_map()
Note: If this argument is a dummy_property_map object, then the color map argument must be specified. For example, adjacency_list with VertexList=listS does not have an internal vertex_index property, so the default value of this argument will be a dummy_property_map object.
Category: UTIL/OUT
Boost.Parameter keyword: boost::graph::keywords::_orig_to_copy
Old-style signature: orig_to_copy(Orig2CopyMap c)
Description: This maps vertices in the original graph to vertices in the copy.
Default: An iterator_property_map created from a std::vector of the result graph's vertex descriptor type, and of size num_vertices(graph), and using the index map argument for the index map.

Complexity

The time complexity is O(V + E).
